本大纲提供了在联合国大会委员会撰写决议草案的标准格式。按照此结构创建符合联合国惯例的正确格式决议。

## 标题信息

每份决议草案都以包含基本信息的标题开头：

### 标题

**(a)** 为您的决议创建一个描述性或创意性的标题。

**(b)** 缩写词很受欢迎，可以使决议令人印象深刻。

**(c)** 示例：*TGIF（打击全球非法金融）* 或 *BREATHE（消除空气中有毒物质的更好法规）*

### 委员会名称

**(a)** 包含委员会的全称。

**(b)** 示例：*联合国大会* 或 *经济及社会理事会*

### 提案国

**(a)** 列出决议的主要作者。

**(b)** 最多：委员会成员的5%，向上取整。

**(c)** 示例：*德国、法国、日本、巴西*

### 签署国

**(a)** 列出希望将决议提交大会辩论的代表。

**(b)** 最少：委员会成员的20%，向下取整。

**(c)** 作为签署国不要求投票支持该决议。

**(d)** 示例：*加拿大、澳大利亚、韩国、墨西哥、肯尼亚*

### 议题

**(a)** 说明您的决议所涉及的议题。

**(b)** 示例：*应对气候变化与环境可持续性*

## 序言性条款

序言性条款为您的决议提供背景和上下文。它们解释为什么这个问题重要以及之前做了什么。

### 格式

**(a)** 每条以*斜体*现在分词（以-ing结尾的动词）开头。

**(b)** 每条以**逗号**结尾。

**(c)** 条款**不编号**。

### 内容

序言性条款通常包括：

**(a)** 对过去联合国行动和决议的引用

**(b)** 承认问题的范围和严重性

**(c)** 认可现有努力及其成果

**(d)** 对情况表示关切或赞赏

### 常用序言性短语

| 短语 | 使用场合 |
|------|----------|
| *承认* | 认可事实或情况 |
| *确认* | 确认原则或承诺 |
| *对……感到震惊* | 对严重情况表示关切 |
| *意识到* | 注意到重要的背景信息 |
| *铭记* | 考虑相关因素 |
| *相信* | 表达对原则的信念 |
| *确信* | 表达对结果的信任 |
| *深切关注* | 强烈表达担忧 |
| *强调* | 突出重要性 |
| *充分意识到* | 着重承认 |
| *遵循* | 遵循既定原则 |
| *审议后* | 在审查信息之后 |
| *注意到* | 观察到一个事实 |
| *察觉到* | 注意到情况或趋势 |
| *重申* | 重申对过去立场的承诺 |
| *认识到* | 承认一个重要事实 |
| *回顾* | 引用过去的决议或条约 |
| *考虑到* | 考虑相关因素 |
| *欢迎* | 对进展表示赞同 |

### 示例序言性条款

> *认识到*各国、人道主义组织和联合国为受危机影响的人们提供援助、住所和医疗服务所做的努力，

## 执行性条款

执行性条款是行动项目——委员会实际希望发生的事情。这些是您决议的核心。

### 格式

**(a)** 每条以**粗体**或*下划线*的第三人称现在时动词开头。

**(b)** 条款**编号**（1、2、3……）。

**(c)** 子条款使用字母（a、b、c），然后是罗马数字（i、ii、iii）。

**(d)** 每条以**分号**结尾，最后一条以**句号**结尾。

### 内容

执行性条款通常包括：

**(a)** 具体的拟议行动和政策

**(b)** 设立委员会、基金或机制

**(c)** 向成员国或联合国机构提出的请求

**(d)** 实施细节和时间表

**(e)** 筹资机制

### 重要说明

大多数联合国委员会（安理会除外）不能强制执行行动。使用*请求*、*鼓励*、*建议*和*敦促*等动词，而非*要求*或*命令*。

### 常用执行性短语

| 短语 | 力度 | 使用场合 |
|------|------|----------|
| *呼吁* | 强 | 敦促各国采取具体行动 |
| *决定* | 最强 | 做出明确的委员会决定 |
| *要求* | 非常强 | 紧急情况（谨慎使用） |
| *鼓励* | 中等 | 建议自愿行动 |
| *支持* | 中等 | 支持现有倡议 |
| *设立* | 强 | 创建新机制或机构 |
| *进一步邀请* | 温和 | 额外建议 |
| *注意到* | 温和 | 承认但不行动 |
| *宣布* | 强 | 发表声明 |
| *建议* | 中等 | 建议行动方案 |
| *请求* | 中等 | 要求采取具体行动 |
| *强烈谴责* | 非常强 | 表达严厉不满 |
| *支持* | 中等 | 支持现有努力 |
| *敦促* | 强 | 推动重要行动 |

### 示例执行性条款

> 1. **呼吁**所有成员国增加对可再生能源发展的承诺；
>
>    a. 鼓励发达国家向发展中国家提供技术援助；
>
>    b. 请求所有国家向秘书长提交年度进展报告。

## 撰写有力决议的技巧

**(1)** **具体明确**：包含具体数字、时间表和责任方。

**(2)** **切合实际**：提出联合国实际能力范围内的行动。

**(3)** **解决资金问题**：解释您的建议将如何获得资金支持。

**(4)** **考虑实施**：谁做什么，何时做，如何做？

**(5)** **争取支持**：包含能吸引多方观点的想法。

**(6)** **检查格式**：确保所有条款遵循正确的格式规范。
