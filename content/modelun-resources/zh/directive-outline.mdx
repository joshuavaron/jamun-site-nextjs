指令是危机委员会中相当于决议的文件。与联合国大会决议相比，指令更简短、更快速、更注重行动。本大纲将帮助你撰写有效的指令。

## 什么是指令？

**(a)** 指令是危机委员会通过的正式文件，用于采取集体行动。

**(b)** 与联合国大会决议不同，指令在模拟中会真正产生效果。

**(c)** 指令专注于行动性条款——没有序言性条款。

**(d)** 指令通常比联合国大会决议更简短、更直接。

## 标题信息

每份指令都以包含基本信息的标题开头：

### 标题名称

**(a)** 为你的指令创建一个有创意或描述性的标题。

**(b)** 首字母缩略词在危机委员会中很受欢迎。

**(c)** 示例：*DUSTIN（Developing Unforgettable Simulations for Thoughtful International Negotiations，即"为深思熟虑的国际谈判开发令人难忘的模拟"）*

### 委员会名称

**(a)** 包含你所在委员会的名称。

**(b)** 示例：*JAMUN秘书处* 或 *亨利八世国王内阁*

### 提案国

**(a)** 列出指令的主要作者。

**(b)** 提案国数量限制因委员会而异——请咨询你的主席。

**(c)** 示例：*Dustin、Joshua、Emma*

### 附议国

**(a)** 列出希望将指令提交大会讨论的代表。

**(b)** 附议国要求因委员会而异。

**(c)** 示例：*Sahana、Sam、Miranda、Abby*

## 行动性条款

指令完全由行动性条款组成——即委员会正在采取的行动。

### 格式

**(a)** 每个条款以**加粗**的一般现在时动词开头。

**(b)** 条款使用**数字**编号（1、2、3...）。

**(c)** 子条款使用字母（a、b、c），然后是罗马数字（i、ii、iii）。

**(d)** 每个条款以**句号**结尾（不像联合国大会决议使用分号）。

### 常用行动性动词

| 动词 | 使用场景 |
|------|----------|
| **创建** | 建立新的组织、项目或结构 |
| **指示** | 命令采取特定行动 |
| **设立** | 建立新的机构或机制 |
| **调动** | 调配资源、军队或人员 |
| **呼吁** | 请求他人采取行动 |
| **授权** | 授予许可或权力 |
| **部署** | 派遣部队或资源 |
| **拨款** | 分配资金或资源 |
| **任命** | 指定人员担任职位 |
| **命令** | 下达特定行动指令（语气较强） |

### 行动性条款示例

> **1. 创建**一个新的情报机构，负责收集关于敌方动向的信息。
>
> **(a)** 该机构将直接向委员会汇报。
>
> **(b)** 该机构被授权：
>
> **(i)** 在敌方领土招募特工；
>
> **(ii)** 在主要城市建立安全屋。
>
> **2. 指示**财政部拨款50万金币用于战争行动。

## 快速参考

| 要素 | 格式 | 示例 |
|------|------|------|
| 标题 | 创意名称或缩略词 | DUSTIN |
| 委员会 | 委员会名称 | JAMUN秘书处 |
| 提案国 | 主要作者 | Dustin、Joshua、Emma |
| 附议国 | 希望讨论该指令的代表 | Sahana、Sam、Miranda |
| 行动性条款 | **加粗**动词、编号、句号结尾 | **1. 创建**... |

## 与联合国大会决议的区别

| 特征 | 联合国大会决议 | 危机指令 |
|------|----------------|----------|
| 序言性条款 | 有 | 无 |
| 条款结尾 | 分号（最后一条用句号） | 句号 |
| 长度 | 较长，更详细 | 较短，更直接 |
| 效果 | 建议采取行动 | 实际产生效果 |
| 动词时态 | 第三人称现在时（呼吁） | 一般现在时（创建、指示） |
| 语气 | 外交性 | 行动导向 |

## 撰写有效指令的技巧

**(1)** **具体明确**：包含关于应该发生什么、谁负责以及涉及哪些资源的具体细节。

**(2)** **注重行动**：指令应该产生实际效果，而不仅仅是表达意见。

**(3)** **考虑后果**：思考你的行动可能如何影响危机的发展。

**(4)** **切合实际**：提出委员会实际权限和资源范围内的行动。

**(5)** **针对当前危机**：专注于眼前的局势，而不是长期规划。

**(6)** **与盟友协调**：与你的联盟合作，确保指令获得支持。
